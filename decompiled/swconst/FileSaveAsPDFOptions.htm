SOLIDWORKS API Help

System Options > Export > PDF

To display the dialog:

Click Tools > Options > System Options > Export > PDF in File Format.

- or -

  1. Click File > Save As.
  2. In Save as type, select Adobe Portable Document Format.
  3. Click Options.

Integer value
