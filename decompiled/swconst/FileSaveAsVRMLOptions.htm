SOLIDWORKS API Help

System Options > Export > VRML

This topic contains two tables. The information in the table:

  • appearing immediately after the screen capture corresponds to the settings on that dialog.
  • titled Obsolete Enumerators contains enumerators that previously appeared on the dialog but are now obsolete.

To display the dialog:

Click Tools > Options > System Options > Export > VRML in File Format.

- or -

  1. Click File > Save As.
  2. In Save as type, select VRML.
  3. Click Options.

Output as - Unit

ISldWorks::SetUserPreferenceIntegerValue(swUserPreferenceIntegerValue_e.swExportVrmlUnits, swLengthUnit_e.<Value>)

Any of the following swLengthUnit_e values:

  • 0 = swMM (millimeters)   

  • 1 = swCM (centimeters)  

  • 2 = swMETER

  • 3 = swINCHES

  • 4 = swFEET
