SOLIDWORKS API Help

System Options > Import > VRML or 3MF

To display the dialog:

Select Tools > Options > System Options > Import > File Format > VRML or 3MF.

- or -

  1. Select File > Open.
  2. In Files of type, select VRML or 3D Manufacturing Format.
  3. Click Options.

  • 0 = swMM (millimeters)   

  • 1 = swCM (centimeters)  

  • 2 = swMETER

  • 3 = swINCHES

  • 4 = swFEET
