SOLIDWORKS API Help

System Options > Colors

This topic contains two tables. The information in the table:

  • appearing immediately after the screen capture of the dialog corresponds to the settings on that dialog.
  • titled Obsolete Enumerators contains enumerators that previously appeared on the dialog but are now obsolete.

 

NOTE: The input or output value is the corresponding IColorTable value unless otherwise specified.
