Service Pack 0
  • New functionality
  • New interfaces
  • New delegates, methods, and properties
  • Obsoleted methods
New functionality
  • Access the configuration-specific custom property manager of:
    • a cut list. See ICutListItem::CustomPropertyManager and IConfiguration::GetCutListItems.
    • an assembly component. See IComponent2::CustomPropertyManager.
  • Retrieve errors that occurred during the last call to IFeatureManager::CreateFeature. See IFeatureManager::GetCreateFeatureErrors.
  • Untrim or extend a surface outside selected edges, optionally trimming inside selected edges. See IFeatureManager::InsertUntrimSurface2.
  • Insert BOM tables in parts, assemblies, and drawings with detailed cut lists and specify whether to dissolve components in indented BOMs. See IBomFeature::DissolvePartLevelRows, IModelDocExtension::InsertBomTable4, and IView::InsertBomTable5.

Other major enhancements

  • Get and set whether to display dual unit values in dimension range lengths of Gtols. See IGtol::GetDisplayDualDimensionInRangeValues and IGtol::SetDisplayDualDimensionInRangeValues.
  • Get the diameter of the sphere encompassing the bounding box of a model. See IModelDocExtension::GetSphericalBoxDiameter.
  • Set whether to save as previous version. See IAdvancedSaveAsOptions::SaveAsPreviousVersion.
  • Insert all model annotations or all reference geometry into a drawing document's selected drawing view. See IDrawingDoc::InsertModelAnnotations4.
