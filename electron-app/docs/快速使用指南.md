# 快速使用指南

## 🎯 全局搜索替换

**在哪里：** 项目面板顶部（项目打开后显示）

**快速步骤：**
1. 输入搜索内容
2. 输入替换内容
3. 可选：勾选"正则表达式"或"区分大小写"
4. 点击"搜索"查看匹配结果
5. 点击"全部替换"执行

**示例：**
- 搜索：`_old` → 替换：`_new`
- 搜索（正则）：`零件\d+` → 替换：`Part`

---

## 🏷️ 设置命名规则

**在哪里：** 虚拟文件夹右键 → "设置命名规则"

**快速步骤：**
1. 开启"启用命名规则"开关
2. 从可用组件中拖拽组件到模板框
3. 为前缀/后缀组件输入值
4. 查看右侧预览效果
5. 可选：勾选"立即重命名现有文件"
6. 点击"保存并应用"

**拖拽组件说明：**
- 🏷️ **前缀/后缀** - 固定文本，需要输入
- 🔢 **序号** - 自动编号（001, 002...）
- 📅 **日期** - 当前日期（20251212）
- 👤 **作者** - 系统用户名

**拖拽技巧：**
- 按住组件拖动到模板框
- 在模板框内可以拖动调整顺序
- 点击 ❌ 删除不需要的组件
- 点击 🗑️ 清空整个模板

---

## 📦 批量重命名文件

**方法1：** 设置命名规则时勾选"立即重命名现有文件"

**方法2：** 虚拟文件夹右键 → "批量重命名"（打开命名规则对话框）

**流程：**
```
右键文件夹
  ↓
选择"批量重命名"或"设置命名规则"
  ↓
构建命名模板
  ↓
查看预览列表（显示前10个）
  ↓
勾选"立即重命名现有文件"
  ↓
保存并应用
```

## 💡 常见用法示例

### 示例1：添加项目前缀
```
拖拽: [前缀: PRJ_]
结果: 零件.SLDPRT → PRJ_零件.SLDPRT
```

### 示例2：序号命名
```
拖拽: [前缀: Part_] → [序号]
结果: 零件.SLDPRT → Part_001.SLDPRT
      装配.SLDASM → Part_002.SLDASM
```

### 示例3：日期+作者
```
拖拽: [作者] → [日期]
结果: 零件.SLDPRT → Administrator_20251212.SLDPRT
```

### 示例4：完整命名
```
拖拽: [前缀: PRJ_] → [日期] → [序号] → [后缀: _V1]
结果: 零件.SLDPRT → PRJ_20251212_001_V1.SLDPRT
```

## 🔧 高级技巧

### 技巧1：序号配置
在模板中添加序号组件后，底部会显示序号设置：
- **起始序号**：从几开始（默认1）
- **位数**：补零位数（默认3位，如001）

### 技巧2：自动重命名
勾选"添加文件时自动按规则重命名"后：
- 新添加的文件自动检查命名规则
- 不符合规则的文件自动重命名
- 无需手动操作

### 技巧3：临时禁用规则
只需关闭"启用命名规则"开关即可：
- 规则配置保留
- 不再自动检查
- 随时可以重新启用

### 技巧4：组合使用
先用全局搜索定位问题文件，再用批量重命名统一修正：
1. 全局搜索找到所有问题文件
2. 记下文件所在文件夹
3. 对该文件夹设置命名规则
4. 立即批量重命名

## ⚠️ 注意事项

1. **先预览后执行** - 确认预览无误再点确定
2. **备份重要文件** - 重命名会修改实际文件
3. **注意文件引用** - SolidWorks装配体可能需要更新引用
4. **序号会递增** - 批量操作时序号自动+1
5. **检查前后对比** - 预览列表会显示所有变化

## 🆘 常见问题

**Q: 为什么有些文件没有重命名？**  
A: 检查文件是否在文件夹内，或者命名规则是否启用

**Q: 序号位数不够怎么办？**  
A: 在序号设置中增加"位数"值（最多6位）

**Q: 可以撤销重命名吗？**  
A: 目前不支持撤销，建议操作前先备份

**Q: 日期格式可以改吗？**  
A: 目前固定为YYYYMMDD格式（如20251212）

**Q: 模板可以保存吗？**  
A: 模板随项目保存，下次打开自动加载
